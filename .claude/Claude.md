# Boutique Trend - MVP Copywriting Assistant

## Descripción
Asistente de copywriting diseñado para una boutique de moda minimalista. Permite generar textos estructurados para diferentes canales a partir de las características de una prenda.

## Entradas (Formulario)
El formulario cuenta con 6 campos para detallar la prenda:
1. **Prenda**: Tipo de prenda (ej. Vestido, Pantalón, Blusa).
2. **Marca**: Nombre de la marca.
3. **Material**: Composición o tejido (ej. Algodón, Seda, Lino).
4. **Color**: Tonalidad principal.
5. **Estilo**: Estilo de la prenda (ej. Casual, Formal, Noche).
6. **Contexto**: Ocasión de uso o temporada (ej. Verano, Evento especial).

## Salidas (Resultados)
El sistema genera tres bloques de texto adaptados a diferentes plataformas:
1. **Web**: Descripción detallada y persuasiva para la ficha de producto en la tienda online.
2. **SEO**: Meta título y meta descripción optimizados para motores de búsqueda.
3. **Instagram**: Copy atractivo para redes sociales, incluyendo hashtags relevantes.

## Requisitos de la Interfaz
- Diseño a dos columnas (Formulario a la izquierda, Resultados a la derecha).
- Estética moderna, minimalista y elegante (colores neutros).
- Botón para copiar al portapapeles en cada bloque de salida.
- Simulación de carga (spinner) de 2 segundos antes de mostrar los resultados.
